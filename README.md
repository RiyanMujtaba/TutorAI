# TutorAI 🧠

An AI-powered school tutor with voice input and output. Ask questions by speaking or typing and get clear, student-friendly explanations powered by Groq AI.

## Features
- Voice input via Web Speech API
- Text-to-speech responses
- Groq AI (Llama 3) for smart, subject-specific explanations
- Supports Maths, Science, History, and more
- Clean, simple UI suitable for students

## Tech Stack
- HTML / CSS / JavaScript (Vanilla)
- Web Speech API (speech recognition + synthesis)
- Groq API (Llama 3.3 70B)

## Getting Started

```bash
git clone https://github.com/RiyanMujtaba/TutorAI
```

Open index.html in Chrome (best browser for Web Speech API). Add your Groq API key where prompted.
